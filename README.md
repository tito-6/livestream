# The Sports Oasis - Frontend

This is the frontend application for The Sports Oasis, a real-time eSports live streaming platform.

## Getting Started

### Prerequisites

- Node.js 18+
- npm

### Installation

1. Navigate to the frontend directory:
   ```bash
   cd frontend
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

### Running the Application

Start the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Features

- **Homepage**: Featured live streams, game categories, and tournaments.
- **Live Stream Player**: Immersive video player with real-time chat and polling.
- **Sports & Schedule**: Browse sports categories and upcoming match schedules.
- **RTL Support**: Full Arabic interface with right-to-left layout.
